Perguntas Frequentes

Encontre respostas para suas perguntas sobre o Web Cloner

O Web Cloner funciona melhor com sites estáticos ou sites com conteúdo dinâmico simples. Alguns sites com proteções anti-scraping, autenticação complexa ou conteúdo altamente dinâmico baseado em JavaScript podem ter limitações.

Clonar sites para uso pessoal, educacional ou arquivamento geralmente é aceitável. No entanto, redistribuir, republicar ou usar comercialmente conteúdo clonado sem permissão pode violar leis de direitos autorais. Use esta ferramenta de forma responsável e respeite os termos de serviço dos sites originais.

Os sites clonados ficam armazenados em sua conta até que você decida excluí-los. Recomendamos baixar os clones para armazenamento local como backup adicional.

O Web Cloner captura HTML, CSS, JavaScript, imagens e outros recursos estáticos. A ferramenta tenta manter a funcionalidade original do site na medida do possível, mas recursos dependentes de APIs externas ou servidores específicos podem não funcionar nos clones.
